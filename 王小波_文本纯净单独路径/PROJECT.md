# 王小波文风训练素材工程

> 创建: 2026-05-23 | 位置: `/home/wy/code1/wangxiaobo/王小波_文本纯净单独路径/`

---

## 一、项目目的

训练 AI 模仿王小波文风——循循善诱、亲切自然、幽默中带讽刺、用思维讲道理。

## 二、历史教训

上一版 `王小波作品整理版/` 失败原因：格式混乱、划分错误、重复素材、缺乏理解层。核心教训：**拆素材之前先做理解**。因此本版在每个素材旁先生成理解辅助文档（作品大纲 + 段落注释），再进行训练数据生成。

## 三、目录结构

```
王小波_文本纯净单独路径/
├── PROJECT.md                    ← 本文件（唯一文档）
├── 01_杂文/                      ← 92 篇（金标准：9节大纲 + 双重分层注释）
├── 02_长篇小说/                  ← 10 部（简版大纲）
├── 03_中短篇小说/                ← 18 部（简版大纲）
├── 04_序言后记/                  ← 3 篇（简版大纲）
├── 05_片段补充/                  ← 7 个（不做分析）
├── 06_书信/                      ← 2 部（跳过）
├── 07_学术/                      ← 1 部（跳过）
└── 08_未竟稿/                    ← 1 部（跳过）
```

## 四、素材分类与训练价值

| 类型 | 数量 | 训练价值 | 处理策略 |
|------|------|----------|----------|
| 杂文 | 92 | ★★★ 核心 | 金标准：9节大纲 + 双重分层注释 |
| 长篇小说 | 10 | ★★☆ 辅助 | 简版大纲（叙事结构+文风特征） |
| 中短篇小说 | 18 | ★★☆ 辅助 | 简版大纲 |
| 序言后记 | 3 | ★★☆ 辅助 | 简版大纲 |
| 片段/补充 | 7 | ★☆☆ 参考 | 跳过 |
| 书信/学术/未竟稿 | 4 | ★☆☆/✗ | 跳过 |

> **核心判断**：杂文才是真正的训练素材——它们展示了王小波"循循善诱讲道理"的核心风格。小说是叙事作品，逻辑结构和杂文完全不同，只需简版大纲提取叙事文风特征。

## 五、辅助文件标准

每个杂文子目录含三个文件：

| 文件 | 内容 |
|------|------|
| `原文.md` | 纯文本，UTF-8 Markdown |
| `作品大纲.md` | 9节结构：整体结构→段落划分→主线逻辑→思维链→主题关键词→叙事技巧→文风特征→人物/案例关系→训练方向 |
| `段落注释.md` | 双重分层：段落层（功能/关键词群/逻辑推进）+ 句子层（逻辑/修辞/文风/训练价值） |

## 附录：王小波简介

王小波，1952 年生于北京。1968 年云南插队，1978 年考入中国人民大学，1984 年赴美匹兹堡大学，1986 年获硕士学位。1988 年回国，先后在北京大学、人民大学工作。1992 年辞去教职成为自由撰稿人。1997 年 4 月 11 日因心脏病突发辞世，年仅 45 岁。

代表作：《黄金时代》《白银时代》《青铜时代》（"时代三部曲"），杂文集《沉默的大多数》《我的精神家园》等。
