# 三十而立 - 作品大纲

## 一、整体结构

```
青春回忆(35%) ⇄ 成年现实(40%) → 和解收束(25%)
```

### 逻辑推进
```
王二三十岁,大学教师→回忆少年:板砖打人,毒杀阿姨→现实:管实验室,抓许由迟到
  → 婚姻:和小胡的日常→意识到自己已变成"好人"
    → "社会是个大熔炉,可以改造各种各样的人,甚至王二"
      → 不甘心的收束:"无论如何,我也不能相信"
```

## 二、段落划分

### 青春回忆

**少年王二**: 书包塞板砖打人→爬烟囱骂老师→幼儿园毒杀阿姨(高锰酸钾+石灰+脚丫泥)
- 功能: 建立王二的"前传"——一个无法无天的野孩子
- 文风: 「王式自嘲」——"帮助后进生王二"成为同学们的固定好事来源

**幼儿园排粪**: 阿姨打毛衣忘了,王二给全班小朋友擦屁股→"今日的窈窕淑女,竟被我捷足先登,光顾了屁股"
- 功能: 全文最荒诞的童年记忆——性、屎、权力的混合

### 成年现实

**实验室**: 许由迟到,臭球鞋烤在烘箱里→"我主!几乎熏死"
- 功能: 王二现在是"管人的"→从被管到管人
- 关键词: 许由、实验、微生物

**婚姻**: 和小胡的日常——吵架、做爱、和解
- 功能: 成年生活的另一面——稳定但乏味

### 收束

**"我不能相信"**: 反复出现的句子——从爬烟囱的少年到赶前四十分钟上班的教师,王二始终"不能相信"自己变成了现在这个样子
- 功能: 三十而立=接受自己是成年人了,但保留"不能相信"的内心

## 三、主题

三十而立=从反叛少年变成社会人。"社会是个大熔炉"——但王二的内心始终有一部分"不能相信"。立住了,但不甘心。

## 四、文风特征

| 特征 | 示例 |
|------|------|
| 回忆交错 | 少年→成年反复切换 |
| 王式自嘲 | "帮助后进生王二" |
| 荒诞童年纪实 | 毒杀阿姨配方 |
| 身体/性直白 | "光顾了屁股" |
