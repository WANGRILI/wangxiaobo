# 中国知识分子与中古遗风 - 作品大纲（金标准版）

## 一、整体结构
```
谁是知识分子(15%) → 中古对比(20%) → 当代批判(40%) → 出路建议(25%)
```

## 二、段落划分

### 第一层：谁是知识分子？（段落1）
- 功能：打破"知识分子"的既有定义，为全文建立更精确的标准
- 中国定义：受过某种程度教育——太宽（知识青年也算？）
- 美国定义：在大学/研究部门、批判社会——太窄（中国几乎没有）
- 最终采用：有一份闲差或教职——可比较的尺度

### 第二层：中古知识分子的对比（段落2）
- 中国中古：县学/州学的读书人→等科举→学伦理学/道德哲学
- 欧洲中古：教士/修道士→拉丁文→神学（一个针尖上能立几个天使）
- 两边都有远大理想：这边以天下为己任，那边献身上帝
- 近代就不像了——中国知识分子保留了中古遗风

### 第三层：中古遗风的表现（段落3-6）

**遗风一：受约束（段落3）**
- 柯老说知识分子"懒、贱，三天不打尾巴翘天上"——学官的嘴脸
- 对比：福柯公开同性恋，法国人没拿他点天灯

**遗风二：论说是非——道德批判（段落4）**
- 中国知识分子关注社会=论说伦理道德
- 外国知识分子关注社会=以科学/理性为基础
- 教皇都笑眯眯的——中国作家动不动"警告×××"

**遗风三：说别人无知（段落5）**
- "连郭鲁茅巴都不知道"——自编的黑话当知识
- "格调低下"=匈牙利小说《会说话的猪》里的腔调
- 批判火力集中在学生和断了腿的同类——"别人咱也惹不起"

**遗风四：树道德模范（段落6）**
- 赞美《渴望》里的完美小媳妇——封建遗风
- 赞美拒绝国外高薪回国——"想要崇高，先要搞到一份高薪聘请以便拒绝掉"
- 想造道德体系——但只呼吁不干："要如何如何"（上帝句式）

### 第四层：路在何方？（段落7-8）
- 中国知识分子的传统："自以为道德清高，地位崇高，有资格教训别人"
- 结果："心比天高，命比纸薄"
- 建议：忘掉"士"的传统——按现代标准：有什么我知道别人不知道的？有什么我想通了别人想不通的？
- 终极判断："虽然缺少了中国特色，但也未见得不好"

## 三、主线逻辑

```
定义知识分子（比较的尺度）
→ 中古时代：中外相似（都以道德/神学为业，都有远大理想）
→ 近代：中国保留了中古遗风——教训别人/论说是非/树道德模范
→ 外国知识分子：以科学和理性为基础/价值中立
→ 出路：忘掉"士"的传统——讲硬道理
```

## 四、核心概念

1. **中古遗风** — 中国知识分子以道德裁判自居的传统
2. **论说是非** — 赤膊上阵做价值评判——v.s. 以理性为基础讨论
3. **"要如何如何"** — 上帝句式——只呼吁不干的姿态
4. **士的传统** — 自以为道德清高、有资格教训别人

## 五、文风特征

1. **中外对比贯穿**：每一层都先描述外国怎么做，再对比中国怎么做
2. **具体案例的杀伤力**："郭鲁茅巴"、"会说话的猪"、《渴望》小媳妇
3. **上帝句式的拆穿**："要如何如何"="上帝说，要有光"——把宏大句式溯源到神学
4. **温和收束**："虽然缺少了中国特色，但也未见得不好"
