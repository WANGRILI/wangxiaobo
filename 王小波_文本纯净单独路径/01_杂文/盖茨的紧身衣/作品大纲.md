# 盖茨的紧身衣 - 作品大纲

## 一、整体结构
```
VR紧身衣设想(25%) → 小说的危机(25%) → 小说的出路(25%) → 杜拉斯(25%)
```

## 二、段落划分与功能

### 段落1：VR紧身衣——技术模拟真实
- 功能：用盖茨的技术预言建立"技术可能取代小说"的威胁
- 紧身衣=完全模拟视觉+听觉+触觉+嗅觉
- 提问："到了那一天，不知人们还有没有心思阅读文本"

### 段落2：法国新小说——上一次技术危机
- 功能：用电影时代小说的危机做类比
- "托尔斯泰花几十页写出的东西，用宽银幕电影几个镜头就能解决"
- 法国新小说的回应：小说应该写内心感受，不是外部场景

### 段落3：小说的韵律
- 功能：提出小说的核心特质——"韵律"
- 反对昆德拉"小说和音乐同质"——但承认有道理
- "小说该写人内在的感觉……还要使这些感觉组成韵律"

### 段落4：杜拉斯的《情人》
- 功能：全文真正目的——纪念杜拉斯，谈什么是好小说
- "它写出一种人生的韵律。书中的性爱和生活中别的事件，都按一种韵律来组织"
- 终极宣言："现代小说有这样的杰作，人若不肯看小说，那是人的错，不是小说的错"
- "既不羡慕电影的镜头，也不羡慕比尔·盖茨的紧身衣"

---

## 三、主线逻辑

```
VR技术可能取代小说 → 但电影也没取代小说 → 
小说的核心是"韵律"(不是图像) → 杜拉斯的《情人》=韵律的杰作 → 
有韵律的小说不怕任何技术
```

---

## 四、核心概念

1. **紧身衣** — 技术模拟的极致，威胁文本的生存
2. **韵律** — 小说的不可替代性——内在感觉的时间组织
3. **杜拉斯** — 证明小说可以不羡慕任何技术的典范

---

## 五、文风特征

1. **技术想象+文学批评**：比尔盖茨和杜拉斯对话
2. **跨越对比**：托尔斯泰→法国新小说→廊桥遗梦→杜拉斯
3. **终极宣言**："人若不肯看小说，那是人的错，不是小说的错"
