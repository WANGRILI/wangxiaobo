# 关于崇高 - 作品大纲

## 一、整体结构
```
捞木头事件(20%) → 脚本比喻(20%) → 崇高批判(30%) → 煽情伦理学(20%) → 罗素对照(10%)
```

## 二、段落划分与功能

### 段落1：一根木头值不值一条命
- 功能：用一个具体事件建立"崇高可能不合理"的问题
- 知青下水捞电线杆→淹死→成烈士
- "我们的一条命，到底抵不抵得上一根木头？"
- 困惑者被批判——"幸好还没有说我反动"

### 段落2：脚本比喻——两种人
- 功能：用"编剧/演员"比喻揭示社会的权力结构
- "自打孔孟到如今，我们这个社会里只有两种人。一种编写生活的脚本，另一种去演出这些脚本"
- 前一种：圣贤/政工干部；后一种：老百姓/知青
- 自定位："我只适合当演员，不适合当编剧，但是看到脚本编得太坏时，总禁不住要多上几句嘴"

### 段落3：崇高并不总是对的
- 功能：全文核心论点
- "崇高并不总是对的，低下的一方有时也会有些道理"
- "有些崇高是人所共知的虚伪，这种东西比堕落还要坏"
- 核心要求："不能只顾浪漫煽情，要留有余地；不能够只讲崇高，不讲道理"

### 段落4：孟子的煽情vs罗素的功利
- 功能：中西对比——两种伦理学
- 孟子：亲亲敬长→奉献一切→崇高——"不讲道理：臣民奉献了一切之后，靠什么活着？"
- 七十年代："为公前进一步死，强过了为私后退半步生"——"我们都死了，谁来干活呢？"
- 核心洞察："高调加虚伪才能构成一种可行的生活方式"
- 罗素：功利主义"虽显得卑下，却关心同胞们的福利"
- 结论：煽情的人应该去写诗——把伦理让给讲道理的人

## 三、主线逻辑

```
捞木头值不值？→ 谁在写脚本？→ 脚本太坏可以提意见 → 
崇高不总是对的 → 虚伪的崇高比堕落还坏 → 
孟子煽情不讲道理 vs 罗素卑下但关心人 → 让讲道理的来做伦理
```

---

## 四、核心概念

1. **脚本比喻** — 社会生活被少数人编写、多数人演出
2. **崇高不总是对的** — 反常识的核心判断
3. **高调加虚伪** — 揭示道德高调的生存论基础
4. **煽情vs讲道理** — 两种伦理学的本质区分

---

## 五、文风特征

1. **从具体到抽象**：一根木头→整个伦理学
2. **脚本比喻贯穿**：编剧/演员——把社会权力结构戏剧化
3. **罗素作为外援**："卑下但关心同胞"vs"崇高但无视利益"
4. **收束建议**："煽情的人应该去写诗"——把对手温和地赶出领域
