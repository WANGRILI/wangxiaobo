# 东西方快乐观区别之我见 - 作品大纲（金标准版）

## 一、整体结构
```
梁任公vs罗素(10%) → 正反馈/自激理论(25%) → 孟子的仁义自激(20%) → 中国的忠孝自激(25%) → 出路(20%)
```

## 二、段落划分

### 段落1：东西方快乐观的对立
- 西方人沉迷物欲，东方人精于人际关系
- 梁任公说西方人有病（列强打仗）→要给他们上课
- 罗素接触东方文明→也觉得有教益
- 现在：东方人还想上课——但没有听众

### 段落2：正反馈（自激）——西方病的根源
- 用控制论的"正反馈"解释：舒服→更舒服→无限循环→自激
- "西方社会正在自激，舒服了还要更舒服，搅到最后，连什么是舒服都不清不楚"
- 海豚实验：刺激快乐中枢→狂敲电键→抽疯

### 段落3：孟子的替代方案——仁义
- 孟子：不要利，要仁义——亲亲敬长很快乐，又不毁坏什么
- 但仁义也会自激："乐则生矣，生则恶可已也……不知足之蹈之手之舞之"
- 王小波："谁要说这不叫抽疯，那我倒想知道一下什么是抽疯"

### 段落4：中国的忠孝自激
- 孝道正反馈：孝子吃香→更孝→走火入魔（郭解埋儿、卧冰求鱼）
- 节烈正反馈：守身如玉被盛赞→割鼻子拉耳朵
- 文革忠字舞：九亿人像发了四十度高烧——孟子的手舞足蹈

### 段落5：人口自激
- "我们的文化重人，所以人多了一定好"→十二亿人→柴禾把山林砍光→地不够种
- "把大家变到身高三寸……就这么缩下去，一直缩到风能吹走"

### 段落6：真正的幸福
- 罗素："人得到自己希求的东西就是幸福"——不是因为幸福才希求
- 需要本身只有一分，非把它弄到十分→过分来自社会的起哄
- "东方社会有东方的起哄法，西方有西方的起哄法"
- 出路："需要什么就去争取什么；需要满足之后，就让大家都得会儿消停"

---

## 三、核心概念
1. **自激** — 控制论概念→文化分析——正反馈失控
2. **起哄** — 社会对个人需要的放大——"干什么事都要别人说好"
3. **东西方的同构** — 物质自激和道德自激是一样的病
