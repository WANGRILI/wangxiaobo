# 京片子与民族自信心 - 作品大纲

## 一、整体结构
```
京片子自豪(20%) → 港台腔入侵(25%) → 文化批判(25%) → 民族自尊心(20%) → 收束(10%)
```

## 二、段落划分与功能

### 段落1：京片子的荣耀
- 功能：建立京片子的文化地位——"北京乃是文化古都，语音也是所有中国话里最高尚的一种"
- 华裔教授被售票员骂——"目瞪口呆，连人家说什么都没有去想"
- 结语："粗话的词汇量不亚于矿工——这不要紧，语音还是我们高贵"

### 段落2：港台腔的入侵
- 功能：对比——京片子的衰落和港台腔的泛滥
- "嗯嗯啊啊的港台腔调""那么来那么去""好可爱好高兴噢"
- 判断："台湾人造的新词新话……国语口音纯正的人学起来很难听"

### 段落3：语言中的民族自尊
- 功能：从语言到文化——港台腔=金钱崇拜
- "我们的一些时髦小姐糟塌自己的语音，肯定是在意淫港币和新台币"
- 解决方案（幽默版）：乒乓球拍子→擀面杖→狼牙棒——逐级升级的治疗
- 核心痛心："北京的女孩子，干嘛要用鼻甲来说话！"

### 段落4：文化的本质
- 功能：从语音上升到文化——什么是真正的文化
- "所谓文化，乃是历朝历代的积累。你把城墙拆了，把四合院扒了，它还在人身上保留着"
- 前国军上校的故事：被北京小姐骂死——"被正庄京片子给骂死，也算是死得其所"

### 段落5：民族自尊心的黄金时代
- 功能：八十年代初的北京小姐——不媚外的尊严
- "就是洛克菲勒冒犯到她，也是照骂不误"
- "别以为有几个臭钱就能在我这儿起腻，惹急了我他妈的拿大嘴巴子贴你！"
- 对比现在：合资饭店小姐不骂人了——教授也不来了

---

## 三、主线逻辑

```
京片子=语音的高贵(历史积累) → 港台腔=金钱的崇拜(文化自卑) → 
真正的文化=身上的燕赵古风 → 八十年代=不媚外的尊严 → 
如今=满地鸟语→教授不来了→我们总该留点东西
```

---

## 四、思维链

1. 用幽默讲述京片子的"高贵"——华裔教授被骂还觉得好听
2. 用愤怒描述港台腔的入侵——从播音员到歌星到饭店小姐
3. 用"乒乓球拍子"递进治疗——从幽默滑向愤怒
4. 转到深层：文化不是建筑是"身上的东西"
5. 前国军上校的故事——极端案例证明京片子的魅力
6. 八十年代vs现在——从"照骂不误"到"满街鸟语"
7. 结尾："我们总该留点东西，好让别人仰慕啊"

---

## 五、主题关键词

1. **京片子** — 既是语言也是文化自信的象征
2. **港台腔** — 金钱崇拜的语言表现
3. **民族自尊心** — 全文真正的主题
4. **仰慕** — 什么东西值得别人仰慕？

---

## 六、文风特征

1. **语言学的幽默自恋**："语音还是我们高贵"
2. **暴力递进**：乒乓球拍→擀面杖→狼牙棒——治理港台腔的"治疗方案"
3. **极端案例**：被骂死的国军上校——"死得其所"
4. **京骂引用**："拿大嘴巴子贴你"——既是批评稿也是京片子展示
