# 对中国文化的布罗代尔式考证 - 作品大纲（金标准版）

## 一、整体结构
```
萧伯纳引子(5%) → 驴变人的荒诞(25%) → 效率计算(15%) → 布罗代尔考证(20%) → 文化根源(20%) → 结论(15%)
```

## 二、段落划分

### 段落1：萧伯纳的耳光
- 萧伯纳剧本：爱尔兰人"一辈子都在弄他的那片土，那只猪，结果自己也变成了一块土，一只猪"
- "我看了这段话，脸上也有点热辣辣"——自省的开端

### 段落2：驴子消失——人取代驴
- 姥姥在时：四十户人家，一百多条驴
- 我去时：一百多户人家，零条驴
- 没有驴以后，人用独轮车推粪——"比驴有了优越性"（反讽）
- 三百斤粪推到二三百米高的山上——早上往返十趟
- 推了两天：满嘴胆汁味、布鞋后跟豁开、小腿肌肉不停震颤、丢脸地接受照顾

### 段落3：效率计算——人不如驴
- 做功一样多——但人吃粮食，驴吃草
- 一斤白薯干=两毛钱；十斤草=两毛钱
- "我不认为自己在吃下一斤白薯干之后，可以和吃了十斤干草的驴比赛负重"
- 白薯干是猪的口粮——"这种可怜的动物后来就改吃人屙的屎"
- 结论：四十户+一百条驴=符合省钱的经济规律；一百多户+零条驴=符合就业的经济规律——两种"经济规律"

### 段落4：布罗代尔的考证——不是个案
- 布罗代尔发现：十六世纪中国的役畜就比西欧少得多，对水力风力利用也不充分
- "这种生存竞争不光在我老家存在，不限于人和驴之间，更不限于七十年代"
- 英国圈地运动=羊吃人；中国=人吃驴——"而且是货真价实的吃"

### 段落5：文化根源
- 中国文化是一种"人本的文化"——但只认为生命是好的，没把快乐、幸福、生存状态定义在内
- "只要大家都能活着就好，不管他们活得多么糟糕"
- 结果：和风力水力比赛推动磨盘，和牲口比赛运输——"比赛谁更不知劳苦，更不贪图安逸"
- 陈永贵和驴比赛负重——驴摔倒，人赢了——"一个伟大的胜利"

### 段落6：结论
- "我们把枪口对准了自己，发明了种种消极的伦理道德：吃苦、耐劳、存天理灭人欲"
- 莎士比亚："苦和累，你吃下的越多，它就越有——所以两者都是无穷无尽的"
- "中国文化对于物质生活的困苦，提倡了一种消极忍耐的态度，不提倡用脑子想，提倡用肩膀扛；结果不但是人，连驴和猪都深受其害"
- 布罗代尔论十七世纪中国："潜在的贫困无处不在"——"我们的祖先怎么感觉不出来？大概是觉得那么活着就不坏吧。"

---

## 三、核心概念

1. **人取代驴** — 不是进步——是用更贵的劳力替代更便宜的畜力
2. **比赛谁更不知劳苦** — 中国文化把"吃苦"变成了竞技
3. **两种经济规律** — 省钱的规律（用驴）vs就业的规律（用人）
4. **潜在的贫困** — 布罗代尔的终极诊断
