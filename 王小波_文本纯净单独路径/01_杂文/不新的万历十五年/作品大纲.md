# 不新的万历十五年 - 作品大纲（金标准版）

## 一、整体结构
```
黄仁宇的书(15%) → 万历的乱糟糟(20%) → 数目字管理(15%) → 岳母与钉子(25%) → 儒学的罐子(25%)
```

## 二、段落划分

### 段落1：推荐一本老书
- "旧的好书总比新的烂书好"——宁愿说不新的《万历十五年》
- 黄仁宇以万历十五年为横断面，剖开中国传统社会

### 段落2：万历的乱糟糟
- 北京城哄传皇上要午朝→所有官员挤到宫门口→皇上自己不知道
- 军队很多但都是废物→极少倭寇席卷半个中国
- 结论："仅靠儒家的思想管理一个国家是不够的……中国必须从一个靠尊卑有序来管理的国家，过渡到靠数目字来管理的国家"

### 段落3：数目字的陷阱
- "若说中国用数字来管理就会有前途，这个想法未免太过天真——数数谁不会呢"
- 大跃进亩产三十万斤——这也是数目字——但比没有数字更糟
- "数字可以是假的，尤其是阿拉伯数字，在后面添起0来太方便"

### 段落4：岳母与钉子——善良不能弥补粗疏
- 黄仁宇说儒家原则本意善良——王小波不同意前半句
- 岳母拿来又粗又钝的钉子要钉进水泥墙→女婿解释钉不进→"你连试都不试？"
- 结果：墙上满是坑——"成了感情和权威的牺牲品"
- "按儒家的标准……符合仁的要求……结果在墙上打了些窟窿"
- "善良要建立在真实的基础上，所以让我去选择道德的根基，我愿选实事求是"

### 段落5：儒学的罐子
- 古往今来的读书人"觉得自己懂了春秋大义，站出来管理国家，结果把一切都管得一团糟"
- "大明帝国是他们交的学费，大清帝国又是他们交的学费"
- "老百姓说：罐子里养王八，养也养不大。儒学的罐子里长不出现代国家来"

---

## 三、核心概念
1. **数目字管理** — 黄仁宇的理想——但数字可以是假的
2. **善良不能弥补粗疏** — 钉子寓言的核心
3. **儒学的罐子** — 老百姓的俗语——精准打击
