# 智慧与国学 - 作品大纲（金标准版）

## 一、整体结构
```
驴和马比喻(15%) → 国学批判(35%) → 智慧的本质(30%) → 终极对比(20%)
```

## 二、核心意象
**驴和马**：草原上的驴（西方智慧）闯入马群（中国国学）→马炸群→驴撵不走→杀/阉/配骡都失败→只能共存。

## 三、段落划分

### 段落1：驴马寓言
- 功能：用一个草原故事建立全文核心隐喻
- 驴=西方智慧（长耳朵怪物/来自内地/善良但陌生）
- 马=中国国学（没见过驴/以为魔鬼/被吓散）
- 结局："有驴子的地方，马就养不住"——马的意见最正确

### 段落2-4：国学批判
- 国学的问题：不追求"智慧"，只追求"有用"和"正确"
- 王小波的核心区分：智慧≠知识≠道德——智慧是"思维的乐趣"
- 国学传统：把一切学问变成道德教化→窒息思维

### 段落5-7：智慧的本质
- 罗素的智慧观：不计利害地追求真理
- 中国的"智慧"：总是被要求"有用"——要么修身齐家治国平天下，要么是"精神原子弹"
- 核心判断：真正的智慧不需要"有用"——它的价值在于它本身

### 段落8：终极对比
- 西方：为智慧而智慧→科学/哲学/艺术
- 中国：为"有用"而智慧→道德文章/经世致用
- 结局：驴和马能否共存？不知道——但马必须接受驴的存在

## 四、主线逻辑
```
驴马比喻（西方智慧vs中国国学）
→ 国学的问题（只求有用不求真）
→ 智慧的本质（不计利害的追求）
→ 中国的"智慧"（修身齐家治国平天下=功利化）
→ 能否共存？（无解之问）
```

## 五、主题关键词
1. **驴和马** — 西方智慧vs中国国学的象征
2. **不计利害** — 罗素对真正智慧的定义
3. **有用** — 中国对智慧的唯一衡量标准
4. **思维的乐趣** — 王小波自己的智慧观

## 六、文风特征
1. 寓言贯穿——驴和马从头到尾
2. 罗素作为外援——借西方哲人说中国问题
3. "思维的乐趣"——王小波的标志性概念
