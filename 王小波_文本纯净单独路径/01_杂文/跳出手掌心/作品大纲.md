# 跳出手掌心 - 作品大纲（金标准版）

## 一、整体结构
```
斯诺两种文化(10%) → 手掌心=自己的领域(25%) → 跳出去(35%) → 中国的困境(30%)
```

## 二、核心意象
**手掌心**：自己熟悉的领域/舒适区。科学家困在自己领域里，文学家困在自己领域里。真正的知识分子应该"跳出手掌心"——进入自己不懂的领域，接受他人的评判标准。

## 三、段落划分

### 段落1：斯诺的两种文化
- 科学知识分子 vs 文学（人文）知识分子
- 互相看不懂、互相看不起
- 王小波的立场：两种文化应该沟通——但不是互相恭维，是互相评判

### 段落2-4：手掌心
- "手掌心"=自己的领域=自己说了算的地方
- 中国的知识分子：每个人都有自己的"手掌心"——在自己的领域里是权威
- 问题：从来不出手掌心——只接受自己领域内的评判
- 结果：学问变成自说自话——没有人敢跨领域评判

### 段落5-7：跳出去
- "跳出手掌心"=接受他人的评判标准
- 科学家应该把自己的成果给文学家评判（能不能说清楚？）
- 文学家应该把自己的想法给科学家评判（有没有事实依据？）
- 核心挑战：你敢不敢让别人（外行）来评判你的工作？

### 段落8-10：中国的困境
- 中国知识分子最大的问题：每个人都守着自己的手掌心
- 结果：科学变成了谁也看不懂的东西（自说自话）——人文学变成了谁也看不懂的东西（自说自话）
- 跳出手掌心=接受普遍的理性标准——在中国几乎没人做到
- 马寅初算一个——他用人口数据说话，不在乎是不是在自己的"领域"

---

## 四、核心概念
1. **手掌心** — 自己熟悉的领域/舒适区/自说自话的权力圈
2. **跳出去** — 接受陌生领域的评判标准
3. **普遍的理性标准** — 不分学科——事实和逻辑
