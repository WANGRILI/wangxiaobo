# 有关"伟大一族" - 作品大纲（金标准版）

## 一、结构
```
老同学重逢(20%) → 什么是伟大(25%) → 梦想不是空想(25%) → 伟大一族在消失(30%)
```

## 二、核心
- 伟大=让梦想成真。盖茨的梦想：让人人都能用计算机。马丁·路德·金的梦想：种族平等
- 不是梦想都是伟大——鲁迅写的那种"呕半口血由丫环扶着看梅花"就不是
- "伟大一族不是空想家，不是只会从众起哄的狂热分子"
- "这一族的人数是越来越少了，将来也许会像恐龙一样灭绝掉"
- "必须准备梦想，准备为梦想工作"

## 三、关键句
> "梦想虽不见得都是伟大事业的起点，但每种伟大的事业必定源于一种梦想。"
