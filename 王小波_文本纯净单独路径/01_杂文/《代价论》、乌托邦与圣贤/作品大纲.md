# 《代价论》、乌托邦与圣贤 - 作品大纲（金标准版）

## 一、整体结构
```
《代价论》评论(20%) → 乌托邦批判(35%) → 圣贤/哲人王(25%) → 收束(20%)
```

## 二、段落划分

### 段落1：中庸不是分析方法
- 郑也夫《代价论》——中庸思想的推广
- 王小波的反驳：中庸是古代人的感觉工具（蒸馒头不发酸），不是近代的分析方法（测PH值）
- "我不以为中庸的思想有任何高明之处"

### 段落2：乌托邦是错误不是激励
- 乌托邦的三个问题：极端国家主义/僵化/乏味
- 郑也夫说乌托邦可以"激励人们向上"——王小波不同意
- 核心判断："乌托邦是前人犯下的一个错误"
- "用自己一次的思想，代替千秋万代后世人的思想"——一个人替所有人思考

### 段落3：哲人王=圣贤
- 从乌托邦上溯到柏拉图→莱库格斯→哲人王
- 罗素的批判："对权势的爱好总是使他们误入歧途"
- "哲人王是什么？就是圣贤啊"——把西方概念钉在中国传统上

### 段落4：我们的乌托邦经历
- "我们曾经经历过乌托邦鼓舞出的蓬勃朝气，只可惜那是一种特殊的愚蠢而已"
- 不是"理想没实现"——而是"理想本身就是错误的"
- 终极判断：不需要一个人替所有人想——"亿万人智力的推动"才可能让社会稍微过得去

## 三、主线逻辑
```
中庸≠分析（古代感觉≠现代方法）
→ 乌托邦=错误（一个人替所有人想）
→ 乌托邦不能激励向上（我们经历过——是愚蠢）
→ 哲人王=圣贤=想替所有人想的人
→ 亿万人智力推动＞一个人设计
```

## 四、核心概念
1. **分析方法vs中庸感觉** — 现代理性vs古代直觉
2. **一个人替千秋万代想** — 乌托邦的本质错误
3. **特殊的愚蠢** — "蓬勃朝气"的真实面目
