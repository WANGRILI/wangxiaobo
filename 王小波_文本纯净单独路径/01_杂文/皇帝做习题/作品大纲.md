# 皇帝做习题 - 作品大纲

## 一、整体结构

### 结构层次
```
历史趣事引入(30%) → 个人共鸣展开(15%) → 核心论证(35%) → 历史回响(20%)
```

### 逻辑推进
```
康熙做几何题 → 我猜他做了→亲近感 → 解几何题的体验(对了知道对错了知道错) →
对比孟子(自以为是) → 计算机同理 → 西学能成事的原因 →
中学为体西学为用=单相思 → 以其昏昏使人昭昭 →
回到历史：康梁不行→康熙来希望大些(假设他做过习题)
```

---

## 二、段落划分与功能

### 第一层：历史趣事——康熙学几何

**段落1前半** - 传教士教皇帝几何
- 功能：建立历史场景，引入核心意象"皇帝做习题"
- 关键词：洋人传道士、欧氏几何学、定理、习题、做没做
- 叙事手法：引述历史文献(张诚日记)，制造"有据可查"的实感

**段落1后半** - "我猜他做了"
- 功能：从"日记没记载"到"我猜做了"——主观判断建立情感连接
- 关键词：做了没有、我猜他做了、试一试、亲近之感
- 思维特征：用猜测建立共鸣——"人家给你出了题目，会不会的总要试一试"

**段落1收尾** - 比汉族读书人亲近
- 功能：提出一个惊人之论——鞑子皇帝比孔孟程朱更亲近
- 关键词：鞑子、皇帝、亲近、隔我们远、没解过几何题
- 对比手法：我们(解过几何题) vs 汉族读书人(没解过) → 康熙(可能解过) → 亲近

---

### 第二层：核心论证——客观对错的力量

**段落2前半** - 解几何题的体验
- 功能：描述"解几何题"的独特心理体验
- 关键词：对了知道对、错了知道错、高兴、郁郁寡欢、实在的成就、不会毫无意义
- 思维特征：从个人体验上升到认知哲学——客观验证给人实在感

**段落2中段** - 比尔·盖茨与计算机
- 功能：平行类比——计算机编程和几何题的共通性
- 关键词：计算机、变压器式的大家伙、着了迷、奇妙之处
- 类比逻辑：几何题=计算机→对了知道对错了知道错→干干脆脆

**段落2中段** - 对比孟子
- 功能：用孟子"养浩然正气"代表中国传统的主观自信
- 关键词：养吾浩然正气、觉得自己事事都对
- 对比：客观验证(对错分明) vs 主观自信(自以为对)

**段落2收尾** - 中学为体西学为用=单相思
- 功能：文章核心批判
- 关键词：单相思、不对、不敢讲、以其昏昏使人昭昭
- 论证：西学能成事是因为有客观验证→想用人家的成果但不接受人家的方法→行不通

---

### 第三层：历史回响——回到康熙

**段落3** - 康梁变法 vs 康熙变法
- 功能：把前面的论证应用到中国近代史
- 关键词：变法、康梁、六君子、技术性的工作、迂夫子、康熙
- 结论：康梁倡导变法够分量，领导变法不行——因为不懂技术性工作
- 回到起点：如果康熙来领导，希望大些——因为皇帝做过习题

## 三、主线逻辑

### 显性主线（叙事线）
```
传教士教康熙几何 → 日记没写做没做 → 我猜做了 → 解几何题的感觉 →
比尔·盖茨玩计算机 → 中学为体西学为用的谬误 → 康梁变法不行 → 康熙来就好了
```

### 隐性主线（哲理线）
```
客观验证(几何题/计算机) → 实在的成就 → 能成事 →
主观自信(孟子养气) → 自以为对 → 不能成事 →
中学为体西学为用 = 想要结果不要方法 → 行不通
```

### 两条线的交汇点
- 康熙做习题：既是一个历史故事(显性)，也是"接受客观验证"的象征(隐性)
- "干干脆脆"：几何题和计算机的共同特征，也是中西思维差异的关键

## 四、思维链模板

### 王小波本篇的典型思维路径

```
历史轶事 → 个人共鸣 → 体验分析 → 跨域类比 → 文化批判 → 历史回响
```

```
步骤1: 从历史文献中找到一个有趣的具体细节
  - 张诚日记记载了传教士给康熙留习题
  - "做没做没有记载"——这个空白正好给作者发挥空间

步骤2: 用自己的经验填补历史空白
  - "我猜他做了"——因为"会不会的总要试一试"
  - 这是一种同理心推理：把自己的经验投射到历史人物身上

步骤3: 分析这种经验的本质
  - 解几何题的体验：对了知道对，错了知道错
  - 核心洞察：这是一种"客观验证"的体验

步骤4: 找平行案例强化论点
  - 计算机编程：同样的干干脆脆
  - 比尔·盖茨的例子证明这不是个例

步骤5: 用对比凸显差异
  - 孟子养气：自以为对——没有客观验证
  - 几何题vs浩然正气——两种认知方式的对比

步骤6: 把差异上升到文明层面
  - "中学为体西学为用=单相思"
  - 你想用人家的成果但不用人家的方法→不行

步骤7: 回到历史，用最初的故事收束
  - 如果是康熙领导变法而不是康梁→希望大些
  - 因为皇帝做过习题——他体验过"客观验证"
```

### 核心概念链
```
康熙做习题 → 解几何题的体验 → 客观验证(对了知道对) → 
计算机同理 → 西学能成事 → 中学为体西学为用不行 → 康熙>康梁
```

## 五、主题关键词

### 一级关键词
1. **客观验证** — 几何题/计算机的核心特质：对错分明
2. **实在的成就** — 解对一道题带来的确定感
3. **干干脆脆** — 西学方法论的概括
4. **单相思** — 中学为体西学为用的本质批判

### 二级关键词
1. **亲近** — 对康熙的情感判断(因共同经验)
2. **浩然正气** — 孟子代表的传统主观自信
3. **技术性的工作** — 变法真正需要的能力
4. **迂夫子** — 对康梁等传统读书人的评价

### 三级关键词（叙事细节）
1. **张诚日记** — 历史文献来源
2. **变压器式的大家伙** — 早期计算机的形象
3. **鼓捣** — 盖茨玩计算机的口语化描述
4. **以其昏昏使人昭昭** — 收束全篇的古语引用

## 六、叙事技巧分析

### 1. 历史文献+主观猜测
- 先引张诚日记建立事实感
- "做没做没有记载"——在事实的边界处插入主观猜测
- 这种"事实+猜测"的结构让猜测有了历史重量

### 2. 个人体验打通历史
- "我猜他做了"→"对我马上就有了亲近之感"
- 跨越三百年，用一个共同的体验(解几何题)连接现代作者和古代皇帝
- 这种"体验共鸣"比理性论证更有说服力

### 3. 跨域类比
- 几何题→计算机→孟子养气
- 三个看似无关的领域被"客观验证"这个共同线索串起来
- 类比的力量：一个领域的洞察照亮另一个领域

### 4. 关键词提炼
- "干干脆脆"——把复杂的方法论差异压缩为四个字
- "单相思"——把一个世纪的文明争论压缩为一个情感比喻
- 这种"给复杂问题起个名字"的能力是王小波的绝技

### 5. 循环结构
- 开头：康熙做习题
- 中间：展开论证
- 结尾：回到康熙——"假设皇上做过习题"
- 首尾呼应的同时，首尾之间的内容让"康熙"的含义从"一个有趣的皇帝"变成了"接受客观验证的象征"

## 七、文风特征

### 语言特点
1. **口语化判断**："我猜他是做了的""干干脆脆""这怎么成呢"
2. **古今混搭**："他是个鞑子，又是皇帝"(把满洲人叫鞑子是清朝汉人的口语)
3. **温和但坚决**："我总觉得这是单相思""但谁都不敢讲"
4. **自谦式介入**："历史不是我的本行，但它是我胡思乱想的领域"

### 句式特点
1. **假设句式**："假如不是皇上不是这样的人""要是解得不对"
2. **判断句式**："我总觉得""这道理很明白"
3. **短句重击**："这就成事""这怎么成呢"

### 王小波标志风格
1. **大题小做**：用"皇帝做习题"这个小切口讨论中西文明差异这个大问题
2. **从体验到理论**：从解几何题的个人体验到"中学为体西学为用"的文明批判
3. **自嘲中带锋芒**："胡思乱想的领域"(自嘲) + "迂夫子是做不来的"(锋芒)
4. **历史想象力**：用"假设"把历史人物拉到现代语境中重新审视

## 八、适用训练方向

### 文风模仿重点
1. **历史切口法**：从历史文献中找一个小细节做大文章
2. **体验共鸣法**：用自己的经验连接历史人物
3. **关键词提炼法**：给复杂问题起一个简单好记的名字
4. **循环结构**：开头和结尾呼应，中间展开

### 逻辑模仿重点
1. **跨域类比**：用一个领域的规律解释另一个领域
2. **客观验证**这个核心概念在不同场景中的应用
3. **从个人到文明**的递进论证

### 不适合模仿的点
1. 张诚日记、康梁变法——特定历史知识
2. "中学为体西学为用"——特定思想史概念
3. 孟子养气——需要中国哲学背景
