# "行货感"与文化相对主义 - 作品大纲（金标准版）

## 一、整体结构
```
宋江故事(15%) → 个人行货感(20%) → 文化相对主义的滥用(35%) → 批判的权利(30%)
```

## 二、段落划分与功能

### 段落1：宋江的行货感
- 功能：用《水浒传》故事建立"行货"概念
- 宋江犯法→戴宗管→宋江不给好处→戴宗大怒："你这厮，只是俺手里的一个行货！"
- "行货"=劣等货物/降价处理品——人被降格为可随意处置的物品
- "我看到这则故事时只有十二岁，从此就有了一种根深蒂固的行货感"

### 段落2：个人行货经历
- 十七岁被装上火车运往云南——"身上别了一个标签：屯垦戍边"
- 传统文化的"解释"：普天之下莫非王土——"还不如说我是个行货更直接些"
- 核心：被当作货物最悲惨的不是"被降价"，是"成为货物"这一事实本身

### 段落3：文化相对主义的噩梦
- 西方学术思潮涌入中国——文化相对主义、功能学派成了"我的噩梦"
- 关键翻转：这些理论本是对的——但可以被歪曲滥用
- 宋江对戴宗说"人人生而平等"→戴宗用文化相对主义反驳："咱们这种文化很好，你还是安心当我的行货吧"
- 功能论也可以歪曲：打你也有打你的功能

### 段落4：批判的权利
- 非洲女孩割礼——不能用文化相对主义来辩护
- 核心判断："人生活在一种文化的影响之中，他就有批判这种文化的权利"
- 我批判中国文化不是因为我在外面——而是因为我活在里面
- 终极宣言："人该是自己生活的主宰，不是别人手里的行货"

## 三、主线逻辑

```
宋江=行货（《水浒传》故事）
→ 我也是行货（十七岁被装火车）
→ 传统文化把行货合理化了（普天之下莫非王土）
→ 西方理论被用来加强行货地位（文化相对主义/功能论的歪曲）
→ 人有权批判自己的文化（因为我活在其中）
→ 终极判断：人该是自己的主宰，不是行货
```

## 四、思维链

1. 从文学作品中提取"行货"概念（命名）
2. 用个人经历证明"行货感"的真实性
3. 反转：用"好理论被坏用"拆穿文化相对主义的滥用
4. 区分：文化相对主义是人类学家对外文化的态度——不是让宋公明当行货
5. 终极权利：活在这文化里的人有权批判这文化

## 五、核心概念

1. **行货** — 被降格为可随意处置的物品——全文最核心的命名
2. **行货感** — 意识到自己是"货物"的悲惨感受
3. **文化相对主义的滥用** — 好理论被用来为压迫辩护

## 六、文风特征

1. **概念命名法**："行货感"——从《水浒传》借一个词定义一种普遍的生存状态
2. **戴宗的反驳**：全文最精彩——让压迫者用"文化相对主义"为自己的压迫辩护——让荒谬自己说话
3. **从文学到个人到理论**：三层递进——宋江→我→所有人
