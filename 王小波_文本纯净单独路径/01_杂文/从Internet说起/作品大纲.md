# 从Internet说起 - 作品大纲（金标准版）

## 一、整体结构
```
Internet(15%) → 禁电脑(15%) → 电影电视(15%) → 禁到小说(20%) → 丧钟为谁而鸣(35%)
```

## 二、段落划分

### 段落1-3：假装赞成——禁Internet、禁电脑、禁影视
- Internet："听说网上黄毒泛滥……这些说法把我吓住了。我很赞成限制"
- 电脑：必须禁——"它能干好事也能干坏事"
- 影视：全部禁掉——"文革十年只看八个样板戏不也活过来了嘛"

### 段落4：禁到小说——露出真面目
- "这样禁来禁去，总有一天禁到我身上"
- "电影电视都能禁，为什么不能禁小说？"
- "好吧，我不写作了，到车站上去扛大包"
- 核心翻转：前面所有"赞成禁止"都是反讽——当禁到你头上时你就不赞成了

### 段落5：丧钟为谁而鸣
- 海明威："所有的人是一个整体，别人的不幸就是你的不幸。不要以为丧钟是为谁而鸣——它就是为你而鸣。"
- 德国牧师：起初抓共产党我不说话……最后来抓我，已经没人能为我说话了
- "这里不是纳粹德国，我也不是新教牧师。所以，这些话我也不想记住。"
- 全文最恐怖的收束——假装不想记住——但已经记住了

---

## 三、核心概念
1. **假装赞成** — 全文的叙事策略——假装站在审查者一边，直到审查落到自己头上
2. **丧钟为你而鸣** — 海明威——当你不为别人说话时，轮到你时也没人为你说话
