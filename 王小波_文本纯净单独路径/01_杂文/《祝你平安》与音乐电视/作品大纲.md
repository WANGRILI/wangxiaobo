# 《祝你平安》与音乐电视 - 作品大纲（金标准版）

## 一、整体结构
```
孙悦的MTV(20%) → 不懂的地方(25%) → 看懂的地方(25%) → 自然主义逻辑(30%)
```

## 二、段落划分

### 段落1：疑惑
- 很少看MTV，但看到了孙悦的《祝你平安》
- 那是一首安慰失意情人的歌

### 段落2：不懂的地方
- 爱情歌串进了女教师和聋哑学生——"主旋律是主旋律，男女之情是男女之情，切不可这样胡串"
- 故事仿佛：聋哑学校教师丢下学生去深圳傍大款→回头不忍→唱祝你平安
- "这歌就不叫《祝你平安》，该叫作《鳄鱼的眼泪》"

### 段落3：看懂的地方——自然主义点题
- 结尾出现交通警微笑着做通行手势——"警察同志让你过去，同时一笑，此乃祝你平安之意也"
- 如果这样拍：《Crazy》就要拍疯子，《我的太阳》就要拍陈佩斯的脑袋
- 岳飞《满江红》就要请食人族——"饥餐胡虏肉"
- 麦当娜《Like a virgin》——难道要请大家当一回妇科大夫？
- "我们又不是妇科大夫，看了也看不懂"

---

## 三、核心概念
1. **自然主义点题** — 用画面直译歌词——最低级的MTV手法
2. **胡串** — 把不相干的东西硬拼在一起
