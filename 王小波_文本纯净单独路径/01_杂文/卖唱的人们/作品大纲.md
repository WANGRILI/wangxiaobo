# 卖唱的人们 - 作品大纲（金标准版）

## 一、结构
```
西单盲人(20%) → 伦敦哗众取宠者(15%) → 洛杉矶燕尾服青年(25%) → 维也纳三重奏(40%)
```

## 二、核心
- 西单盲人卖唱："歌唱得过于悲惨"——"凡是他们唱过的歌我再也不想听到"
- 伦敦一人乐队：浑身挂满乐器——"他的演奏没有艺术，就是要钱"
- 洛杉矶：燕尾服青年弹钢琴——"只弹奏一曲，就如向你表示好意"
- 维也纳：三个不到二十岁的音乐学院学生在歌剧院前演奏——"就如童声一样，是一种天籁"
- "我坚信，这一幕是当日维也纳最美丽的风景"
- 结论："青年动人之处，就在于勇气，和他们的远大前程"

## 三、四个案例的递进
```
悲惨（西单盲人）→ 吵闹（伦敦）→ 优雅（洛杉矶）→ 天籁（维也纳）
```
