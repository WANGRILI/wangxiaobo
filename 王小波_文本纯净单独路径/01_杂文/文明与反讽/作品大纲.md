# 文明与反讽 - 作品大纲（金标准版）

## 一、整体结构
```
烤圣徒故事(15%) → 维多利亚假正经(25%) → 反讽=解毒剂(25%) → 福尔斯(20%) → 收束(15%)
```

## 二、段落划分

### 段落1：烤架上的幽默
- 圣徒被烤→说"下面烤好了该翻翻了"
- "烤肉比厨师还关心烹调过程"——反讽的极致
- "只要你不怕做烤肉，就没有什么阻止你说俏皮话"
- "总得有一定程度的文明，才能理解这种幽默——所以，幽默的圣徒就这样被没滋没味的人吃掉了"

### 段落2：维多利亚的假正经
- 英国人不说腰以下部位——钢琴腿用布遮起来
- 鸡胸=白肉，鸡腿=黑肉——"简直是脑子有点毛病"
- 结果：假正经的反面——大量地下色情小说涌现
- "问题还不在于该不该写性，而在于不该写得没劲"

### 段落3：反讽=解毒剂
- "任何一个文明都该容许反讽的存在，这是一种解毒剂，可以防止人把事情干到没滋没味的程度"
- 当一种现象开始"贫"了（犯贫=重复到无趣）——需要反讽来制止
- 福尔斯：把维多利亚地下小说翻写成女孩得肺炎死了——"反对犯贫"
- "文学的使命就是制止整个社会变得无趣"

### 段落4：中国的无趣
- 想向《红楼梦》开火——但不敢——"刚到主编手里，他就要把我烤了"
- 罪名：亵渎文化遗产、民族虚无主义
- 烤圣徒的故事在中国只能改为：圣徒在烤架上高呼"我主基督万岁"直至烤熟
- "连这个故事也变得很没劲了"

---

## 三、核心概念
1. **反讽** — 文明的解毒剂——防止无趣
2. **没滋没味** — 无趣的最高级——比"错"更可怕的评价
3. **犯贫** — 重复到无趣——需要反讽来打断
