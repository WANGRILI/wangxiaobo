# 生命科学与骗术 - 作品大纲

## 一、整体结构
```
科学是什么(20%) → 骗术分析(25%) → 罗素的科学权威论(15%) → 有罪推定(15%) → 民族主义(15%) → 收束(10%)
```

## 二、段落划分与功能

### 段落1：中国人不懂科学
- 功能：建立问题——中国人把科学当巫术
- 许倬云："中国人先把科学当作洪水猛兽，后把它当作呼风唤雨的巫术"
- 核心问题："对不懂科学的人说明什么是科学，就像要对三岁孩子说明什么是性一样"

### 段落2：为什么外行会误解
- 维纳/爱因斯坦的例子："他们说话，不该让外行人听见"
- 山里人买灯泡的故事——看不懂电灯就骂灯泡为什么不亮
- 判断："科学对于公众来说，确实犯下了过于深奥的罪孽"

### 段落3：罗素的科学权威论
- 科学权威和其他权威不同：你可以自己验证
- "科学家声称自己发明、发现了什么，都要主动接受别人的审查"

### 段落4：科学上的"有罪推定"
- 司法=无罪推定；科学=有罪推定
- "我王某人在此声明自己最终证明了哥德巴赫猜想……就等于把自己置于骗子的地位"
- 同理：特异功能者应首先被推定为骗子，直到他们拿出证据

### 段落5：骗术的证明问题
- 特异功能"有时灵有时不灵"——说不出什么时候灵
- 笑话：物理学家致电瑞典："我发现了核聚变方法但现在忘了——先把奖给我"
- 江湖骗子层次太低——只想到混吃混喝和骗稿费

### 段落6：民族主义催生伪科学
- 许倬云："中国人在科学面前，很容易失去平常心"
- "很多人急于发明新东西，为民族争光"
- 大跃进的"超声波"——冷水变热水——割破洗澡者的屁股

### 段落7-8：科学的本质
- "科学是全人类的事业，它的成就不能为民族所专有"
- 罗素："科学在'不计利害地追求客观真理'"
- 最终判断：所谓"生命科学"——"换个名字，到别处去试试吧"

## 三、主线逻辑

```
科学=理性(可验证/接受审查) → 中国人不懂科学(当巫术) → 
特异功能=骗术(有罪推定) → 民族主义催生骗子 → 
科学=全人类事业(不归属任何民族)
```

## 四、核心概念

1. **科学的深奥** — 对公众犯下的"罪孽"
2. **有罪推定** — 科学认定新发现的规则
3. **平常心** — 面对科学应有的态度
4. **超声波** — 大跃进伪科学的标本

1. **笑话拆穿法**："核聚变方法忘了但奖先给我"——用笑话类比骗术
