# 论战与道德 - 作品大纲（金标准版）

## 一、整体结构
```
五十年代逻辑讨论(15%) → 文革论战(15%) → 当下文化争论(25%) → 屎撅子比喻(20%) → 结论(25%)
```

## 二、段落划分

### 段落1：五十年代逻辑大讨论
- 逻辑问题应是对错之争→实际争的是"谁是无产阶级谁是资产阶级"
- "此种论战的方式有恫吓、威胁之意"
- 还算"比较平和的讨论"——没被送往北大荒

### 段落2：反右到文革——论战的升级
- 争谁好谁坏→一旦分出好坏→好人揭露坏人的私生活
- 文革两派：朗诵毛主席著作——意在让对方领会自己是"毒蛇"
- 争不出结果→刀兵相见

### 段落3：当下的文化争论
- 论战方式没变——还是争谁好谁坏
- 《唐明皇》制片人：探讨反对者的民族精神、国学修为、道德水准、智商
- "唯一令人庆幸的是，还没有探讨那些朋友的先人祖宗"
- 看国产影视的危险："看了好电影不觉得好，你就不够好；看了坏电影不觉得坏，你就成了坏蛋"

### 段落4：屎撅子比喻
- "我是个本世纪初年的爪哇土著人——生来勇敢，但更重视清洁"
- "荷兰殖民军和他们作战时，就把屎撅子劈面掷去，使他们望风而逃"
- 审美批评=子弹（不怕）→道德非难=粪便（胆寒）
- "当我和别人讨论文化问题时……道德方面的非难就如飞来的粪便那样使我胆寒"

### 段落5：结论
- 文化界赌的是人品、爱国心、羞耻心——"有点像赌命"
- 争完了就忘了——"从起头上就没有把廉耻当廉耻，道德当道德"
- 呼吁："算了罢，何必呢？"
- 艺术和科学是崇高的事业——但它们不该和个人的道德绑在一起

---

## 三、核心概念
1. **争谁好谁坏** — 中国论战的本质——不是争对错
2. **屎撅子比喻** — 道德非难=粪便——全文最精彩的意象
3. **两个论域** — 审美/科学≠道德——混淆就是"赌命"
