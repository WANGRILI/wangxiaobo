# 积极的结论 - 作品大纲（金标准版）

## 一、整体结构
```
姥姥不信亩产三十万斤(15%) → 积极结论的荒诞(35%) → 消极的自由(25%) → 个人立场(25%)
```

## 二、段落划分

### 段落1：姥姥的常识
- 一亩地打三十万斤粮食→姥姥跳着小脚叫"杀了俺俺也不信"
- 算了一本细账：三十万斤摊在一亩地上有两尺厚
- 全家攻击姥姥"觉悟低"→但姥姥是对的——常识赢了

### 段落2-4：积极结论的荒诞
- 那个年代要求一切都有"积极结论"：形势大好、前途光明
- 即使饿死人也要得出"积极结论"
- 核心荒诞：不是先有事实再有结论——是先有"必须是积极的"再有"事实"
- 万物都要求有积极结论=花剌子模的变体——只能有好消息

### 段落5-7：消极的自由
- 王小波的价值翻转：消极不是坏东西
- "消极的自由"（不被强迫的自由）比"积极的强制"更重要
- 个人体验：在那个年代保持沉默已经是最大的抵抗
- "我没有积极结论，我有的是消极的自由——不说话"

### 段落8：个人立场
- 不需要每件事都得出"积极结论"
- 有些事情没有积极结论——接受这一点的能力就是理性
- 最终自白："我是一个消极的人，但我并不为此感到羞愧"

---

## 三、主线逻辑
```
姥姥不信谎言（常识的胜利）
→ 时代要求一切积极（谎言的制度化）
→ 什么是"积极"？（把坏说成好）
→ 消极的价值（不说话的抵抗）
→ 个人的选择（不需要每件事都积极）
```

## 四、核心概念
1. **积极结论** — 制度化的乐观——把一切说成好的
2. **消极的自由** — 不被强迫说"好"的自由
3. **姥姥的常识** — 全文的精神支点——普通人最朴素的判断力
