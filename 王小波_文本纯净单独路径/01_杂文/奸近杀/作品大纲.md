# 奸近杀 - 作品大纲（金标准版）

## 一、整体结构
```
《廊桥遗梦》争议(20%) → 茶花女=卖淫嫖娼(15%) → 扔石头的小伙伴(20%) → 蚂蚱寓言(25%) → 结论(20%)
```

## 二、段落划分

### 段落1：《廊桥遗梦》的争论让人烦
- 有人说宣扬婚外恋该批判，有人说否定婚外恋不该批判
- 电影和"婚外恋"焊在一起了
- "这是编出来的故事……就算是真的，也是美国人的事"

### 段落2：茶花女=卖淫嫖娼人员
- 巴黎歌剧院来演《茶花女》→中国观众："这个茶花女是个妓女啊！玛格丽特和阿芒，正好是一对卖淫嫖娼人员！"
- "要是小仲马在世，听了这种评价，一定要气疯"
- "从那时到现在，已经过了十几年。我总觉得中国的观众应该有点长进——谁知还是没有长进"

### 段落3：扔石头的小伙伴
- 小伙伴见公鸡踩蛋就扔石头——"要制止鸡耍流氓"
- "这位朋友长着鸟形的脸，鼻涕经常流过河，有点缺心眼"
- "不知为什么，傻人道德上的敏感度总是很高"
- "太古板的人没法欣赏文艺作品，他能干的事只是扰乱别人"

### 段落4：蚂蚱寓言
- 清朝笔记：才子看两只蚂蚱交尾→癞蛤蟆跳出来把两只蚂蚱都吃了
- 作者的感慨："奸近杀"
- 王小波的重新解读："这两只蚂蚱在篱笆底下偷情，是两个堕落分子。而那只癞蛤蟆，却是个道德上的义士"
- "蚂蚱不交尾，就没有小蚂蚱；没有小蚂蚱，癞蛤蟆就会饿死"

---

## 三、核心概念
1. **道德敏感度过高** — 傻人的特征——把一切看成道德问题
2. **奸近杀** — 中国传统：把男女关系极端化——连蚂蚱都不放过
3. **癞蛤蟆不是义士** — "它顶多会想：今天真得蜜，一嘴就吃到了两个蚂蚱"
