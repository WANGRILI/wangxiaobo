# 迷信与邪门书 - 作品大纲

## 一、整体结构
```
知识分层(10%) → 案例展示(25%) → 理论分析(25%) → 批判伪科学(20%) → 知识分子责任(20%)
```

## 二、段落划分与功能

### 段落1：知识的三个层次
- 可靠的知识(根本) → 可验证的猜测 → 想入非非(偶尔)
- 核心：人有能力分开可信和不可信——"但也有例外"

### 段落2：狐仙附体——撒癔症
- 农村妇女自称狐仙附体——"不可信"
- 社会学解释：农村生活艰苦乏味——"认定自己不是女人而是狐狸，也许会愉快一些"
- 判断："这是一种病态"——但我同情她

### 段落3：临终的秘方——走投无路的迷信
- 同学父亲临终指顶棚说有祖传秘方——拆了顶棚也没找到
- 解释："人在胸中抹煞可信和不可信的界限，多是因为生活中巨大的压力"
- 马林诺夫斯基：巫术源于压力

### 段落4：自我假设——我也会迷信
- 如果我得癌症→有人发外气能救我→我会信
- 如果我在奥斯维辛→有人能用意念让希特勒改变主意→我全给
- 核心：对"事出有因的迷信"宽容

### 段落5：巫婆神汉——不能宽容的骗人者
- 区分：撒癔症(病人/值得同情) vs 装神弄鬼(骗子/人渣)
- 邪门书的作者——"凭良心说，我真不知好在哪里"

### 段落6：知识分子的诚信
- "知识分子的道德准则应以诚信为根本"
- 科学家也会被骗——有科学技能未必有科学素质

### 段落7：小说家的诚信
- "小说家以编故事为主业；既知道何谓真，更知道何谓假"
- "利用他人的弱点进行欺诈，干尽人所不齿的行径"——不是小说家该做的事

### 段落8：社会责任
- "中国有'北拳南革'两大祸患"——迷信是"北拳"
- "中国人——尤其是社会的下层——有迷信的传统……简直就是渴望迷信"
- 最终警告："作家应该有社会责任感，不可为一点稿酬，就来为祸人间"

---

## 三、主线逻辑

```
知识分三层(可信→猜测→想入非非) → 
普通人迷信(压力导致、值得同情) → 
职业骗子(趁火打劫、不可宽容) → 
知识分子骗人(诚信破产) → 
北拳传统(中国隐患) → 作家要有责任感
```

---

## 四、核心概念

1. **可信/不可信的界限** — 理性的基础
2. **事出有因的迷信** — 可同情但仍是病态
3. **人渣** — 趁火打劫的骗子
4. **北拳** — 中国迷信传统的隐喻

---

## 五、文风特征

1. **自我假设**："让我得上癌症……我就会信"——用自己为例
2. **奥斯维辛的极端案例**：最极端的压力验证最极端的迷信
3. **北拳南革**：《老残游记》的引用——赋予批判历史深度
