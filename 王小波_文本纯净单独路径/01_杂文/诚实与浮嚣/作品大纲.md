# 诚实与浮嚣 - 作品大纲（金标准版）

## 一、整体结构
```
费孝通与《江村经济》(30%) → 诚实的成色(25%) → is/ought混淆(25%) → 浮嚣批判(20%)
```

## 二、段落划分

### 段落1：费孝通的《江村经济》
- 哥哥被问"中国人写的科学著作哪本最值得读"→答：《江村经济》
- 理由："十分诚实地描述了江南农村的生活景象，像这样的诚实在中国人写的书里还未曾有过"
- 核心价值：把事实写到了纸上——在中国很少有人做这样的事
- 马林诺夫斯基作序也称赞了费先生的诚实——"达到了国际先进水平"

### 段落2：诚实的成色
- "诚实就像金子一样，有成色的区别"
- 国内版叫《江村经济》——成色差一些（迎合中国文人口味）
- 海外版叫《中国农民的生活》——"十足赤金式的诚实"
- 命名本身就反映了诚实的程度问题

### 段落3：is/ought混淆
- 朱熹一辈子搞不清"是"（be）和"应该是"（should be）
- "是"=事实，"应该是"=意愿——两者不能混淆
- 做学问时把意愿当事实——不做学问时又能分开了
- "不独朱夫子，中国人做学问时都是如此"
- 浮嚣的根源：把想要的东西说成已经有的东西

### 段落4：浮嚣批判
- 中国的学术文章：话越说越大、越说越空
- "浮嚣"=用漂亮的空话替代对事实的诚实描述
- 对比：费孝通的《江村经济》——没有一句空话
- 收束：诚实是中国学术最稀缺的品质

---

## 三、核心概念
1. **诚实** — 把事实写到纸上——中国最稀缺的学术品质
2. **is/ought** — 事实和意愿的混淆——浮嚣的根源
3. **成色** — 诚实有程度之分——十足赤金vs掺了水的
4. **浮嚣** — 用漂亮空话替代事实
