# 警惕狭隘民族主义的蛊惑宣传 - 作品大纲（金标准版）

## 一、整体结构
```
罗素的概括(10%) → 希特勒的蛊惑宣传(20%) → 蛊惑宣传的三特征(40%) → 反对蛊惑宣传(30%)
```

## 二、段落划分

### 段落1：蛊惑宣传的力量
- 罗素：人活在世上主要做两件事——改变物体位置和支使别人这样干
- 知识分子还有第三种：蛊惑宣传——"这可是种厉害东西"
- 希特勒《我的奋斗》：每个字都害死了若干人

### 段落2：蛊惑宣传的三个特征

**特征一：半疯不傻**
- "蛊惑宣传不是真话——否则它就不叫作蛊惑——但它也不是蓄意编造的假话"
- 旧俄国疯僧：被狂热信念左右，信口雌黄，人见人怕
- "宁要社会主义的草不要资本主义的苗"——非得像江青用更年期高亢啸叫声说出来才有人信

**特征二：给人快感**
- 纳粹千年帝国、文革跑步进入共产主义——迎合急功近利的人
- 最简单的快感来源：煽动仇恨——"鼓励大家去仇恨一些人、残害一些人"
- "煽动仇恨、杀戮，乃至灭绝外民族，都不要花费什么"
- "我们应该体谅蛊惑宣传家，他们也是没办法"——反讽

**特征三：因正派人士的宽容而得逞**
- 纳粹初起时德国人对它抱有幻想
- 文革初起时老师帮忙发动——"等皮带敲到自己脑袋上时，连后悔都不敢了"
- "有些人像瘾君子盼毒品一样，渴望著新的蛊惑宣传"

### 段落3：反对蛊惑宣传
- "我的抱负只能是反对蛊惑宣传。我别无选择。"

---

## 三、核心概念
1. **半疯不傻** — 蛊惑宣传的语言状态——不是蓄意说谎，是借酒撒疯
2. **仇恨是最便宜的快感** — 煽动仇恨不需要成本
3. **正派人士的宽容** — 蛊惑宣传得以成功的最大推手
