# 我为什么要写作 - 作品大纲（金标准版）

## 一、整体结构
```
登山家答案(10%) → 减熵过程(25%) → 父亲的安全计划(20%) → 危险倾向(20%) → 最终答案(25%)
```

## 二、段落划分

### 段落1：登山家的回答
- "因为那座山峰在那里"——"明明是自己想要登山，偏说是山在那里使他心里痒痒"
- 幽默感——不直接说"我想"，说"山在召唤"
- 热力学角度：登山=减熵现象——人趋利避害（熵增），趋害避利=减熵

### 段落2：写作=减熵
- 别人写作是因为"热"（小说热、诗歌热）——随大流=熵增
- "我的情形不一样。前三种热发生时，我正在美国念书，丝毫没有受到感染"
- 家训不准学文科——"立志写作在我身上是个不折不扣的减熵过程"

### 段落3：父亲的安全计划
- 老舍跳了太平湖，胡风关了监狱，王实味被枪毙了——"他老人家也是屋内饮酒门外劝水"
- 父亲让五个孩子全学理工——"希望我们每个人都学一种外行人弄不懂而又是有功世道的专业，平平安安地度过一生"
- 哥哥的例外：七八年考大学时是北京木城涧煤矿最强壮的青年矿工——"吼起来比我爸爸音量还要大"——所以敢学哲学
- 图哈切夫斯基元帅对萧斯塔科维奇说："我小时候很有音乐天才。只可惜我父亲没钱给我买把小提琴！"——说完了没多久就被毙了

### 段落4：危险倾向
- "我身上总有一股要写小说的危险情绪"
- 插队时写坏领导从尾骨开始变成驴——"后来读了卡夫卡，发现他也写了个类似的故事"
- 美国：American's business is business——永远是经商热
- 哥哥到了美国也后悔——应该学商科
- "别人都有钱挣，自己却在干可疑的勾当，脸面上也过不去"

### 段落5：最终答案
- 人类的存在和文明的发展=减熵过程
- "大家都顺着一个自然的方向往下溜，最后准会在个低洼的地方汇齐，挤在一起像粪缸里的蛆"
- "光有熵增现象不成"
- 终极回答："我相信我自己有文学才能，我应该做这件事。但是这句话正如一个嫌疑犯说自己没杀人一样不可信。所以信不信由你吧。"

---

## 三、核心概念

1. **减熵** — 逆自然而行——写作=对抗世界往下溜的惯性
2. **安全计划** — 父亲让所有人学理工——在会枪毙文人的年代
3. **粪缸里的蛆** — 如果所有人都熵增——最后会挤在一起
