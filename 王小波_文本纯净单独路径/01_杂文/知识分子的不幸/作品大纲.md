# 知识分子的不幸 - 作品大纲（金标准版）

## 一、整体结构
```
乔叟故事(10%) → 什么是知识分子(25%) → 不幸的根源(35%) → 罗素的出路(30%)
```

## 二、段落划分

### 段落1：乔叟故事
- 《坎特伯雷故事集》武士提问"女人最大的心愿是什么"
- 答案："有人爱她"——中世纪的正确答案
- 暗示：每个时代都有自己的"正确答案"——但正确答案不一定是真的

### 段落2-3：知识分子的定义
- 知识分子≠有知识的人——知识分子是以"思考"为业的人
- 核心职能：不是给出"正确答案"，而是不断思考、质疑
- 中国的"知识分子"传统：读书人=给出"正确答案"的人（修身齐家治国平天下）

### 段落4-6：不幸的根源
- 知识分子在中国的不幸：被要求给出"正确"答案而非"真"答案
- 两个例子：冯友兰修改《中国哲学史》、郭沫若的" scientifical "颂诗
- 核心悲剧：最有头脑的人被迫说最没头脑的话

### 段落7-8：罗素的出路
- 罗素：知识分子的职责是"不计利害地追求真理"
- 在中国的困境：追求真理可能"累及自身"
- 正面例子：马寅初——坚持说真话，付出代价
- 反面例子：太多人选择了"安全"而非"真理"

### 段落9：个人收束
- 知识分子的不幸不在于被打倒——在于被迫自己打倒自己
- "我这一生，最大的不幸是生为一个中国人……不，生为一个中国的知识分子"

---

## 三、核心概念
1. **知识分子** — 以思考为业、不计利害追求真理的人
2. **正确答案vs真答案** — 两个"正确"的区别
3. **累及自身** — 说真话在中国的代价
