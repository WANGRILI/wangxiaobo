# 科学的美好 - 作品大纲（金标准版）

## 一、整体结构
```
个人学科学失败史(30%) → 科学是什么(35%) → 科学与人文(20%) → 科学的本质(15%)
```

## 二、段落划分

### 段落1-3：学科学的可笑失败
- 化学：用嘴吸移液管→吸了浓氨水→"像吸到了陈年的老尿罐里"
- 漏氯仿→吸进去能醉死十条大蟒→但站着不倒
- 数学：做题像小日本冲锋→结果只做了一半（没看到背面）
- 自嘲："我和科学没有缘分，但是我爱科学，甚至比真正的科学家还要爱得多些"

### 段落4-6：科学是什么
- 科学≠技术≠知识——科学是一种"不计利害地追求客观真理"的态度
- 罗素的定义贯穿全文
- 科学的美好不在于它的用处——在于它"只是为了知道"
- 中国的问题：总问科学"有什么用"——不问"是什么"

### 段落7-8：科学与人文
- 斯诺的"两种文化"——科学和人文的隔阂
- 王小波的立场：科学的美和人文的美可以相通——都在于"求真"
- "真正的科学家和真正的艺术家是同一种人"

### 段落9：科学的本质
- 科学=人类最诚实的事业——不接受没有证据的说法
- "我爱科学，正如我爱一个不撒谎的人"
- 收束：科学的"美好"在于它的诚实

---

## 三、核心概念
1. **不计利害** — 科学的本质（罗素）
2. **诚实** — 科学和艺术的共通点
3. **用嘴吸移液管** — 全文最精彩的细节——对科学的笨拙之爱
