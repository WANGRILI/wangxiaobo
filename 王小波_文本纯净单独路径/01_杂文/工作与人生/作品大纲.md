# 工作与人生 - 作品大纲（金标准版）

## 一、整体结构
```
人生如一日(10%) → 三种人生主题(25%) → 工作=乐趣(25%) → 心胸(25%) → 活着不是一切(15%)
```

## 二、段落划分

### 段落1：人生如一日
- 童年=清晨、正午=精力充沛但已疲惫、黄昏=沉入休息
- "工作是人一生的主题"

### 段落2：三种人生主题
- 农村：生儿育女——"把儿女养大，自己就死掉，给他们空出地方来"
- 城市：社会地位——八宝山墓碑："副系主任、支部副书记、副教授、某某教研室副主任"——"假如能把这些'副'字去掉个把"
- 美国：生卒年月+服兵役——"这位上帝的子民曾经来到尘世，以及这位公民曾去为国尽忠"

### 段落3：工作=乐趣
- "人从工作中可以得到乐趣，这是一种巨大的好处"
- 写小说=我的目标
- "干什么都是好的，但要干出个样子来，这才是人的价值和尊严所在"
- "人工作时不单要用到手腿和腰，还要用脑子和自己的心胸"

### 段落4：心胸
- "心胸是我在生活中想要达到的最低目标"
- "某件事有悖于我的心胸，我就认为它不值得一做"
- "罗素先生曾言，对人来说，不加检点的生活，确实不值得一过"
- "人必须过他可以接受的生活，这恰恰是他改变一切的动力"

### 段落5：活着不是一切
- "中国人喜欢接受这样的想法：只要能活着就是好的，活成什么样子无所谓"
- "抱有这种想法的人就可能活成任何一种糟糕的样子"
- 终极：某种样子的文章"绝不能让它从我笔下写出来"——这是对生活的态度

---

## 三、核心概念
1. **工作=主题** — 不是谋生手段，是人生的主题
2. **心胸** — 自己能接受的最低标准
3. **活着不是一切** — 反对"只要能活着就好"
