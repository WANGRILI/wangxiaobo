# 明星与癫狂 - 作品大纲

## 一、整体结构
```
美国影迷(20%) → 中国癫狂(25%) → 理论分析(25%) → 明星责任(20%) → 好莱坞批判(10%)
```

## 二、段落划分与功能

### 段落1：美国影迷的癫狂
- 功能：用美国追星场景建立"癫狂"概念
- 艾迪·摩菲影迷——通宵排队买票(其实白天也有)→"有计划有预谋地把自己置于一场癫狂之中"
- 判断："正常人不该是这个样子"

### 段落2：中国的明星癫狂
- 功能：中国的追星比美国更疯
- 对比链：美国(体育场闹)vs中国(送别墅/献身体/客串机长)
- 文革类比："这使我想起了文革初的某些场景"
- 判断："表面上没有美国闹得厉害，实际上更疯得没底"

### 段落3：癫狂无害论——但有前提
- 功能：区分——癫狂本身没问题，误解才有问题
- "人既有这样一股疯劲，把它发泄掉总比郁积着好"
- "把脑子放在家里，到体育场里疯上一阵，回来把脑子装上、再去上班"
- 关键区分："虽然有这种癫狂，大家并没有疯，这一点很重要"

### 段落4：明星不该控制癫狂
- 功能：批判明星的狂妄——以为能控制影迷的癫狂程度
- 女明星涮四川影迷→影迷发火→女明星说"不懂明星制"
- 判断："迷到何种程度该由那些人自己来决定"
- 男明星撞人后要打受害者→"把受害者看做追星的影迷，觉得他该心甘情愿地挨顿揍"
- 结论："追星族不用我们操心，倒是明星，应该注意心理健康"

### 段落5：好莱坞的糖衣炮弹
- 功能：从明星上升到文化——好莱坞的两面性
- 好莱坞"有城府的拳师，会耍花拳绣腿；但也另有真相，不让你看到"
- "明星制"是帝国主义的糖衣炮弹——香港已中弹
- 大陆：有明星没有出色的表演，更没有经典艺术片
- "这个游戏玩到此种程度，已经过了，应该回头了"

## 三、主线逻辑

```
追星=有计划有预谋的癫狂 → 正常人可以发疯但要可控 → 
明星不能替影迷控制癫狂程度 → 中国明星把闹当成了真本事 → 
好莱坞有真功夫也有花拳绣腿 → 我们只学了花拳绣腿
```

---

## 四、核心概念

1. **有计划有预谋的癫狂** — 追星族的本质
2. **脑子放家里** — 癫狂和正常的切换
3. **明星制的误解** — 以为可以控制别人的癫狂
4. **糖衣炮弹** — 好莱坞明星制的文化入侵

---

## 五、文风特征

1. **冷眼旁观**："我一直是这样的冷血动物"
2. **极限假设**：影星客串外科医生/机长——把追星推到逻辑极限
3. **脑子放家里的比喻**：癫狂=暂时不用脑子
