# 人性的逆转 - 作品大纲（金标准版）

## 一、整体结构
```
贫困是生活方式(10%) → 洛克vs孟子(20%) → 学大寨=追求痛苦(15%) → 扁担vs牛顿(20%) → 孟子的混蛋逻辑(15%) → 结论(20%)
```

## 二、段落划分

### 段落1：贫穷是一种生活方式
- 西方发展学者说：贫穷是一种生活方式（有人不想富）
- 推广：贫困是生活方式，富裕也是；追求聪明是态度，追求愚蠢也是
- "在这个世界上，有一些人在追求快乐，另一些人在追求痛苦"

### 段落2：洛克vs孟子——伦理学的根基
- 洛克：人人都追求快乐→趋利避害→伦理学的根基
- 孟子：礼高于利，义高于生→创造了一种复杂的伦理学
- 王小波的判断：前一种对→产生富裕和快乐；后一种错→产生贫困和痛苦

### 段落3：学大寨——追求痛苦的实证
- 劳动超过人体极限→毫无价值→"没别的，就是要给人找些罪来受"
- 干部积极分子乐此不疲
- 结果：更穷——"人干了艰苦的工作之后，就变得很能吃，而地里又没有多长出任何可吃的东西"

### 段落4：扁担vs牛顿——肉身痛苦的哲学根源
- 南方扁担/北方小车→每寸肌肉骨骼在百十公斤压迫下
- 西方：阿基米德→牛顿→卡特→一二百年前就解决了
- 核心原因不是技术——是感情问题："西方人以为，人的主要情感源于自身，所以就重视解决肉体的痛苦。中国人以为，人的主要情感是亲亲敬长，就不重视这种问题"
- 终极判断："从孔孟到如今，中国的哲学家从来不挑担、不推车。所以他们的智慧从不考虑降低肉体的痛苦，专门营造站着说话不腰疼的理论"

### 段落5：孟子的混蛋逻辑
- 孟子说反对他的人（杨朱、墨子）都是禽兽
- 由此得出崇高的定义："有种东西，我们说它是崇高，是因为反对它的人都不崇高"
- 王小波："还不如直说凡不同意我意见者都是王八蛋为好"

### 段落6：结论——一加一等于二
- 乔治·奥威尔："一切的关键就在于必须承认一加一等于二"
- 在扭曲的年代：有人强迫你说一加一不等于二
- "人性的逆转"=从知道一加一等于二变成不知道

## 三、主线逻辑
```
贫穷是生活方式（有人选择痛苦）
→ 洛克：人追求快乐→趋利避害（对）
→ 孟子：礼高于利（错）→产生痛苦
→ 实证：学大寨→追求痛苦→更穷
→ 扁担vs牛顿→降低肉身痛苦vs营造站着说话的理论
→ 崇高的虚伪定义：反对我的人都不崇高
→ 终极：一加一等于二——但有人强迫你否认
```

## 四、核心概念
1. **人性的逆转** — 从追求快乐变为追求痛苦
2. **站着说话不腰疼的理论** — 不挑担的哲学家营造的学问
3. **一加一等于二** — 理性最后的底线
