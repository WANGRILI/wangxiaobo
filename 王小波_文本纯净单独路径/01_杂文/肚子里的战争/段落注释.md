# 肚子里的战争 - 段落注释

| 关键句 | 手法 | 文风 |
|--------|------|------|
| "穿着白大褂的，不叫他大夫又能叫什么呢" | 命名困境 | 用称谓暴露现实的荒诞 |
| "全院的病人都在外面看着，互相打赌说几个小时找到阑尾" | 荒诞全景 | 手术=娱乐节目 |
| "越是不熟就越是要动——在战争中学习战争！" | 语录挪用 | 「王式借用」：毛语录用在找盲肠上——神圣消解 |
| "人的肠子和战争不是一码事，但这话就没人说了" | 朴素道理 | 「王式直击」 |
| "虽然有些刀口偏左，有些偏右，还有一些开在中央，但所有的刀口都开在了肚子上，这实属难能可贵" | 假表扬 | 最低标准的"表扬"=全部荒谬 |
| "急得主刀大夫把他的肠子都拿了出来，上下一通紧倒。清晨时分厨师在门外洗猪大肠，就是这么一种景象" | 降维比喻 | 手术→洗猪大肠 |
| "最爱的猪大肠，自从看过这个手术，再也不想吃了" | 后果收束 | 「王式幽默」 |
| "那个闹事的人是在借酒撒疯。这就是说，他明知道自己在胡闹，但还要闹下去，主要是因为胡闹很开心" | 核心洞察 | 全文哲学落点 |
| "怪不得总吃不饱，原来都漏掉了" | 黑色幽默 | 哥们豪迈的最后一句话——最好的收尾 |
