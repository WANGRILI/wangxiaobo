# 我看国学 - 作品大纲（金标准版）

## 一、整体结构
```
读四书感想(20%) → 孔子(10%) → 孟子(10%) → 朱熹(15%) → 中西对比(20%) → 口香糖比喻(15%) → 钻牛角尖(10%)
```

## 二、段落划分

### 段落1：读四书的总体感想
- 老师说我国学底子不行→发愤读四书到二程朱子
- 孔子："是个挺可爱的老天真"——"那儿有一种匹克威克俱乐部的气氛"
- 至于他的见解，"也就一般"——礼和文革里的仪式差不多

### 段落2：孟子
- "甚偏执，表面上体面，其实心底有股邪火"
- "无君无父，是禽兽也"——"不是一个绅士的作为"
- "如果生在春秋，见了面也不和他握手"

### 段落3：中西对比
- 佩服法拉第、牛顿、爱因斯坦——"人家想出的东西完全在你的能力之外"
- 孔孟的仁义中庸——"似乎都用不着特殊的思维能力就能想出来，琢磨得过了分，还有点肉麻"
- 二程看鸭子体会"仁"——"毛茸茸的鸭子虽然好看，但再怎么看也是只鸭子"
- "我虽然不佩服孔孟，但佩服古代中国的劳动人民。劳动人民发明了做豆腐"

### 段落4：朱熹
- 朱熹用阴阳五行格尽天下万物——"多少有点失之于简单"
- 往井底看能看到一团白气——"还能看到一个人头，那就是你本人"
- "可能观察得不仔细……还有可能是井太深，但我不相信宋朝就没有浅一点的井"

### 段落5：口香糖比喻
- 美国将军给随从嚼了俩钟头的口香糖——"没味道不奇怪，我给你之前已经嚼了两个钟头了"
- "四书五经再好，也不能几千年地念；正如口香糖再好吃，也不能换着人地嚼"
- "那块口香糖再嚼下去，还能嚼出牛肉干的味道，只要你不断地嚼"

### 段落6：钻牛角尖
- "古宅闹鬼，树老成精，一门学问最后可能变成一种妖怪"
- 国学"有人说不所不包，到今天还能拯救世界，虽然我很乐意相信，但还是将信将疑"

## 三、核心概念
1. **匹克威克俱乐部** — 孔子学堂的气氛——亲切但没什么学问
2. **口香糖** — 四书五经被嚼了两千年——该换新的了
3. **做豆腐** — 劳动人民的创造>孔孟的仁义——最经典的价值翻转
