# 《青铜时代》万寿寺 - 作品大纲

## 一、整体结构

现实线(40%) ⇄ 虚构线(40%) → 两线交汇(20%)

### 逻辑推进
"我"在医院失忆→万寿寺→桌上手稿(薛嵩的故事)→边读边改写→现实与虚构互渗→接受:我就是写故事的人

## 二、段落划分

### 现实线

**开篇**: 《暗店街》"我的过去一片朦胧"。医院→万寿寺→手稿
- 功能: 建立失忆前提。元小说框架——小说关于"写小说"本身
- 关键词: 失忆、万寿寺、手稿

**中间**: 读手稿。薛嵩在湘西,酷暑。叙述者不满意→开始改写
- 功能: 嵌套叙事。"我"→手稿→改写→新版本
- 关键词: 薛嵩、红线、铁枪

**收束**: 接受"我是写故事的人"。记忆不是找回的,是重新创造的
- 功能: 身份确认=重新开始写作

### 虚构线: 薛嵩的故事(多版本)

**版本一**: 薛嵩与刺客,红线识破
**版本二**: 薛嵩与红线,主奴关系
**版本三**: 薛嵩与田承嗣冲突
- 功能: 同一故事反复重写=每次都是重新定义自己

## 三、思维链条

```
失忆→找手稿→读/改写→现实虚构互渗→"我是写故事的人"
```

## 四、主题

失忆=失去写作能力。找回记忆=重新开始写作。写作即存在。
"青铜时代"=个人史中被遗忘但渴望被重新发现的时代。
